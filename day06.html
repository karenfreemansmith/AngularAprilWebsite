Design & Layout Project 4 Components

Goals

  • Identify reusable sub-components for project
  • Create sub-components
  • Use routing to send data to another component
  • Improve page layout and design

Notes:

Pass data through a routerLink tag:

  • Modify app.module.ts
    • import { RouterModule, Routes } from '@angular/router';
    • { path: 'profile/:name', component: ProfileComponent },
  • Modify component.ts
    • import { ActivatedRoute } from '@angular/router';
    • detail="";
    • 
  constructor(private route: ActivatedRoute) {
    this.route.params.subscribe(params=>this.detail=params.detail);
  }
  • Add {{detail}} to component.html
  • Modify your router link: [routerLink]="['/details', 'detail']"
