Notes:

Adding Properties to a Component

In component.ts:

  • add "Input" to imports
  • add "inputs: []" to the @Component decorator (with the list of property names you plan to add...)
  • (optional) you can set default values in the export class section

In the component.html:

  • add the property name & value to the component tag (myProperty="myValue")
